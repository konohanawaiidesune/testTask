Данное приложение представляет собой примитивный Проводник.
Оно служит для навигации между каталогами и для открытия файлов.

Путь, по которому необходимо перейти, можно прописать вручную в поле ввода в верху окна.

Чтобы запустить приложение, откройте .ехе файл (test_explorer.exe) в корневой папке проекта.
В директории tests находятся исходные файлы приложения и файлы для тестирования (test_explorer.py).
Файлы upper.qrc и upper.py - ресурсы для иконки кнопки "выше" (переход на один уровень вверх).
_design файлы содержат интерфейс.

Вполне возможно, что на Вашем экране окно отображается маленьким. Это происходит в связи 
с низким разрешением экрана ноутбука разработчика. 

Приложение не может открыть файл, если для него не определена программа, в которой он открывается по умолчанию.
Также не открываются сторонние приложения в силу ограниченных прав (при тестировании не помог и запуск от имени администратора).
